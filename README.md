# BMI + TDEE Calculator

โปรแกรมคำนวณดัชนีมวลกาย (BMI) และปริมาณแคลอรี่ที่ควรได้รับต่อวัน (TDEE)
พัฒนาด้วย HTML, CSS และ JavaScript (Vanilla JS) ล้วน ไม่ใช้ framework หรือ library ภายนอก

## โปรแกรมทำอะไรได้บ้าง
- กรอกส่วนสูง น้ำหนัก อายุ เพศ และระดับกิจกรรมในแต่ละวัน
- คำนวณค่า BMI พร้อมแสดงเกณฑ์ (น้ำหนักน้อย / ปกติ / เกิน / อ้วน)
- คำนวณค่า TDEE (แคลอรี่ที่ควรได้รับต่อวันเพื่อรักษาน้ำหนักปัจจุบัน) ด้วยสูตร Mifflin-St Jeor
- ตรวจสอบความถูกต้องของข้อมูลที่กรอก (validation) เช่น ห้ามปล่อยว่าง ห้ามใส่ค่าติดลบ

## สูตรที่ใช้
- **BMI** = น้ำหนัก (กก.) ÷ [ส่วนสูง (ม.)]²
- **BMR (Mifflin-St Jeor)**
  - ชาย: `10×น้ำหนัก + 6.25×ส่วนสูง − 5×อายุ + 5`
  - หญิง: `10×น้ำหนัก + 6.25×ส่วนสูง − 5×อายุ − 161`
- **TDEE** = BMR × ตัวคูณกิจกรรม (1.2 – 1.9 ตามระดับที่เลือก)
